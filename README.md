# 📱 Lost & Found Management System

A modern **Lost & Found Management System** developed using **Flutter** and **Firebase** to help users report, search, claim, and manage lost or found items efficiently. The system provides a seamless experience for both **users** and **administrators** with real-time updates and an intuitive interface.

---

## 🚀 Features

### 👤 User Features

* 🔐 User Authentication (Login & Register)
* 👤 Profile Management
* 🖼️ Profile Picture Upload (Cloudinary)
* ➕ Create Lost/Found Reports
* ✏️ Edit Reports
* 🗑️ Delete Reports
* 📋 View My Reports
* 🔍 Explore Lost & Found Items
* ❤️ Claim Found Items
* 💬 Real-time Chat System
* 👀 Message Seen Status
* 👤 View Other User Profiles
* 🚫 Block User
* 🚩 Report User
* 🔔 Notification System

---

### 🛠️ Admin Features

* 📊 Admin Dashboard
* 👥 Manage Users
* 📦 Manage Reports
* 📑 Manage Claims
* 📈 Analytics Dashboard
* 🗑️ Delete Reports & Claims
* 🔄 User Role Management
* 📋 Timeline Activity Monitoring
